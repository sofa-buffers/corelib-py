# corelib-py

Pure-Python, zero-dependency runtime for the **SofaBuffers** binary wire format — the Python counterpart to `corelib-c-cpp`, `corelib-rs`, `corelib-go`, `corelib-java` and `corelib-cs`.

Distribution: `sofabuffers` · import package: `sofab` · Python 3.9+.

See the design plan and the C reference test vectors used in `tests/`.
